수강신청 연습 단국대 수강신청 연습 사이트 형태로 로그인, 시간표 조회, 신청 흐름을 직접 체험할 수 있습니다.
동시성 실험 조건부 UPDATE, 락 전략, MySQL 기반 동시 요청 시뮬레이션으로 병목 지점을 추적했습니다.
성능 측정 Micrometer, Prometheus, Grafana, k6를 통해 CPU, 메모리, 응답시간, DB 풀 사용량을 수집합니다.

이 사이트가 하는 일

단국대 수강신청 연습, 수강신청 모의 테스트, 시간표 조회 연습을 중심으로 만든 연습용 사이트입니다. 실제 학사 시스템과는 무관하며, 개인 학습과 성능 실험 목적으로 운영됩니다.

중점적으로 본 문제

  • 동시에 많은 사용자가 신청할 때 정원 초과 없이 처리되는가
  • CPU, 메모리, DB 커넥션 풀 중 무엇이 먼저 병목이 되는가
  • 실제 사용 패턴에서 어떤 지연 구간이 생기는가

주요 기술

  • Spring Boot, JPA, MySQL, HikariCP
  • Micrometer, Prometheus, Grafana
  • k6 부하 테스트, 동시성 제어 실험
